# 🎓 Student Management System

## 📌 Description
This project is a simple system to manage student records. It allows users to add, update, and delete student data. It also calculates grades and stores data in files.

## 🚀 Features
- Add student records
- Update student data
- Delete records
- Calculate grades
- File handling for storage

## 🛠️ Technologies Used
- C Programming
- File Handling
- Structures & Functions

## ▶️ How to Run
1. Compile the code:
   gcc main.c -o sms
2. Run the program:
   ./sms
